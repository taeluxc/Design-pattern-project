# Food Ordering System - Abstract Factory Pattern

## Overview

This project demonstrates the use of the Abstract Factory Design Pattern in a food ordering system.

## Features

* Supports multiple restaurant types (Fast Food, Italian)
* Creates Meal, Drink, and Dessert objects
* Easy to extend with new restaurant types

## Design Pattern

Abstract Factory Pattern

## Project Structure

* factories: contains factory classes
* products: contains product interfaces and implementations

## How to Run

1. Compile the project
2. Run FoodOrderingSystem.java
3. Change factory type to switch restaurant
